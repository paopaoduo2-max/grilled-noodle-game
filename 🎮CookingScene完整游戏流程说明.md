## 🎯 完整游戏流程

### 第一步：放置面饼（直接放到铁板）
1. 点击 `🍞 面饼` 按钮
2. 面饼会**直接出现在铁板中央**
3. 控制台提示：`✅ 面饼已放到铁板上！现在可以添加其他食材了。`

### 第二步：添加鸡蛋
1. 点击 `🥚 鸡蛋` 按钮
2. **左上角显示** `✋ 🥚 鸡蛋`（表示拿在手上）
3. 点击**铁板上的面饼**
4. 鸡蛋会添加到面饼上，出现在面饼上方
5. 控制台提示：`✅ 🥚 鸡蛋 已添加！`

### 第三步：添加香菜
1. 点击 `🧅 香菜` 按钮
2. 左上角显示 `✋ 🧅 香菜`
3. 点击铁板上的面饼
4. 香菜添加到面饼上
5. 控制台提示：`✅ 🧅 香菜 已添加！`

### 第四步：添加洋葱
1. 点击 `🧅 洋葱` 按钮
2. 左上角显示 `✋ 🧅 洋葱`
3. 点击铁板上的面饼
4. 洋葱添加到面饼上
5. 控制台提示：`✅ 🧅 洋葱 已添加！`
6. 控制台提示：`🎉 所有食材都添加完成！点击 [上菜] 按钮完成制作！`

### 第五步：上菜
1. 此时 `✅ 上菜` 按钮变为**亮绿色**（可点击）
2. 点击 `✅ 上菜` 按钮
3. 完成的烤冷面会飞向右侧客人区域
4. 控制台提示：`🎉 烤冷面制作完成！客人很满意！`
5. 1秒后自动重置铁板，可以开始制作下一份

## 🔧 需要手动完成的步骤（约2分钟）

### 步骤1：打开CookingScene
在Cocos Creator中打开 `assets/Scenes/CookingScene.scene`

### 步骤2：绑定CookingController脚本
1. 在**层级管理器**中选择 `Canvas` 节点
2. 在**属性检查器**中点击 `添加组件` → `自定义脚本` → 搜索 `CookingController`
3. 脚本会添加到Canvas上

### 步骤3：绑定属性引用
在 `CookingController` 组件中，将以下节点拖拽到对应属性：

| 属性名称 | 绑定节点 | 位置 |
|---------|---------|------|
| `grillArea` | GrillArea | Canvas/GrillArea |
| `foodContainer` | FoodContainer | Canvas/GrillArea/FoodContainer |
| `handItemLabel` | HandItem | Canvas/HandItem |
| `doughBtn` | DoughBtn | Canvas/IngredientsPanel/DoughBtn |
| `eggBtn` | EggBtn | Canvas/IngredientsPanel/EggBtn |
| `cilantroBtn` | CilantroBtn | Canvas/IngredientsPanel/CilantroBt |
| `onionBtn` | OnionBtn | Canvas/IngredientsPanel/OnionBtn |
| `serveBtn` | ServeBtn | Canvas/IngredientsPanel/ServeBtn |

### 步骤4：保存并测试
1. 按 `Ctrl+S` 保存场景
2. 点击 **预览** 按钮
3. 在MainMenu选择第一关进入游戏

## 💡 交互提示

### 正确操作
- ✅ 第一次点击面饼 → 直接放到铁板上
- ✅ 点击其他食材 → 拿在手上（左上角显示）
- ✅ 点击铁板上的面饼 → 添加手上的食材
- ✅ 所有食材添加完 → 上菜按钮变绿
- ✅ 点击上菜 → 完成制作

### 错误提示
- ❌ 铁板上已经有面饼，再次点击面饼 → 提示 `⚠️ 铁板上已经有食物了！`
- ❌ 铁板上没有面饼，点击其他食材 → 提示 `❌ 请先放置面饼！`
- ❌ 重复添加同一食材 → 提示 `⚠️ XX已经添加过了！`
- ❌ 食材未添加完就上菜 → 提示 `❌ 还有食材没有添加完成！`

## 📝 代码逻辑说明

### 状态管理
```typescript
// 手持物品
private currentHandItem: IngredientType | null = null;

// 铁板上的食物节点
private foodOnGrill: Node | null = null;

// 已添加的食材列表
private addedIngredients: IngredientType[] = [];

// 铁板是否被占用
private isGrillOccupied: boolean = false;
```

### 核心流程
1. **点击面饼** → `placeDoughOnGrill()` → 创建面饼节点
2. **点击其他食材** → 设置 `currentHandItem` → 更新手持显示
3. **点击铁板** → `addIngredientToFood()` → 创建食材子节点
4. **点击上菜** → `serveFood()` → 动画 + 重置

### 动画效果
- **面饼出现**：从小到大的弹跳动画（backOut）
- **食材添加**：先放大再缩小的反馈动画
- **上菜**：飞向客人区域并缩小

## ❓ 常见问题

### Q1：点击按钮没有反应？
**A：** 检查CookingController组件的属性是否都已绑定。特别是按钮引用。

### Q2：食材不显示在铁板上？
**A：** 检查 `grillArea` 和 `foodContainer` 是否正确绑定。

### Q3：上菜按钮一直是灰色？
**A：** 确保所有食材（鸡蛋、香菜、洋葱）都已添加到面饼上。

### Q4：控制台有报错？
**A：** 查看具体错误信息。常见问题：
- 节点引用为null → 检查属性绑定
- 组件获取失败 → 检查节点结构
