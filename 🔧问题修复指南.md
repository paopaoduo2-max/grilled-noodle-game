# 🔧 MainMenu场景问题修复指南

## 问题描述
1. 点击关卡按钮没有反馈
2. 点击"开始游戏"是空界面
3. 无法进入第一关或教程关卡

## 🎯 快速修复步骤

### 修复1：添加关卡选中视觉反馈（5分钟）

**问题**：点击关卡按钮后没有高亮显示，不知道选中了哪个。

**解决方案**：更新MainMenuUI.ts，添加视觉反馈

#### 打开文件：`assets/Scripts/UI/MainMenuUI.ts`

找到 `onSelectLevel` 方法（大约在182行），替换为：

```typescript
/**
 * 选择关卡
 */
private onSelectLevel(levelId: number) {
    this._selectedLevelId = levelId;
    console.log(`[MainMenuUI] 选择关卡: ${levelId}`);
    
    // 添加视觉反馈：高亮选中的关卡
    this.updateLevelButtonSelection();
}

/**
 * 更新关卡按钮选中状态
 */
private updateLevelButtonSelection() {
    if (!this.levelPanel) return;
    
    // 遍历所有关卡按钮
    for (let i = 0; i < this.levelPanel.children.length; i++) {
        const btnNode = this.levelPanel.children[i];
        const sprite = btnNode.getComponent(Sprite);
        if (!sprite) continue;
        
        // 解析levelId
        const levelIdStr = btnNode.name.replace('Level_', '');
        const levelId = parseInt(levelIdStr);
        
        // 如果是选中的关卡，高亮显示
        if (levelId === this._selectedLevelId) {
            sprite.color = new Color(255, 220, 100, 255); // 金黄色高亮
        } else {
            // 恢复原始颜色
            const level = GameConfig.LEVELS.find(l => l.levelId === levelId);
            if (level) {
                if (level.isTutorial) {
                    sprite.color = level.unlocked ? new Color(100, 255, 100, 255) : new Color(150, 150, 150, 255);
                } else {
                    sprite.color = level.unlocked ? new Color(100, 200, 255, 255) : new Color(150, 150, 150, 255);
                }
            }
        }
    }
    
    console.log('[MainMenuUI] 已选中关卡:', this._selectedLevelId);
}
```

### 修复2：确保场景在构建设置中（2分钟）

**问题**：场景可能没有添加到构建列表中。

**解决方案**：

1. 在Cocos Creator菜单栏，点击 **项目 → 构建发布**
2. 在弹出窗口左侧，找到 **参与构建场景**
3. 确保以下场景都被勾选：
   - ✅ MainMenu
   - ✅ PrepareScene
   - ✅ CookingScene
   - ✅ ResultScene
   - ✅ TutorialScene

4. 如果没有勾选，点击 **添加场景** 按钮
5. 找到 `assets/Scenes/` 文件夹
6. 依次添加所有场景
7. 关闭构建窗口（不用实际构建）

### 修复4：添加调试日志（3分钟）

**问题**：不知道代码执行到哪一步了。

**解决方案**：更新 `onStartGame` 方法

在 `assets/Scripts/UI/MainMenuUI.ts` 中，找到 `onStartGame` 方法，替换为：

```typescript
/**
 * 开始游戏
 */
private onStartGame() {
    console.log('[MainMenuUI] ========== 开始游戏 ==========');
    console.log('[MainMenuUI] 选中的关卡ID:', this._selectedLevelId);
    
    const gameManager = GameManager.Instance;
    
    if (!gameManager) {
        console.error('[MainMenuUI] ❌ GameManager不存在！');
        alert('错误：GameManager未初始化');
        return;
    }
    
    console.log('[MainMenuUI] ✅ GameManager存在');
    
    // 检查关卡是否有效
    const level = GameConfig.LEVELS.find(l => l.levelId === this._selectedLevelId);
    if (!level) {
        console.error('[MainMenuUI] ❌ 关卡不存在:', this._selectedLevelId);
        alert(`错误：关卡${this._selectedLevelId}不存在`);
        return;
    }
    
    if (!level.unlocked) {
        console.warn('[MainMenuUI] ⚠️ 关卡未解锁:', level.levelName);
        alert(`关卡未解锁：${level.levelName}`);
        return;
    }
    
    console.log('[MainMenuUI] ✅ 关卡有效:', level.levelName);
    console.log('[MainMenuUI] 调用 gameManager.startLevel...');
    
    gameManager.startLevel(this._selectedLevelId);
    
    console.log('[MainMenuUI] startLevel 调用完成');
}
```

### 修复5：默认选中第一关（1分钟）

**问题**：可能用户没有点击关卡，`_selectedLevelId` 是初始值。

**解决方案**：

在 `assets/Scripts/UI/MainMenuUI.ts` 的 `start()` 方法最后添加：

```typescript
start() {
    console.log('[MainMenuUI] start开始');
    this.createLevelButtons();
    console.log('[MainMenuUI] start完成');
    
    // 添加以下代码：默认选中第一个解锁的关卡
    const firstUnlockedLevel = GameConfig.LEVELS.find(l => l.unlocked);
    if (firstUnlockedLevel) {
        this._selectedLevelId = firstUnlockedLevel.levelId;
        this.updateLevelButtonSelection(); // 如果添加了修复1的代码
        console.log('[MainMenuUI] 默认选中关卡:', this._selectedLevelId);
    }
}
```

### 修复6：检查GameManager节点（2分钟）

**问题**：GameManager可能没有正确初始化。

**解决方案**：

1. 在Cocos Creator中打开 `MainMenu.scene`
2. 在层级管理器中找到 **GameManager** 节点
3. 选中该节点
4. 在Inspector面板中检查：
   - ✅ 节点存在
   - ✅ 节点是激活状态（勾选）
   - ✅ 有 `GameManager` 组件

5. 如果GameManager节点不存在：
   - 在Canvas外层创建新节点，命名为 `GameManager`
   - 添加 `GameManager` 组件
   - 保存场景

import { _decorator, Component, director, Button } from 'cc';
import { GameManager } from '../Manager/GameManager';

const { ccclass } = _decorator;

@ccclass('MainMenuUISimple')
export class MainMenuUISimple extends Component {
    start() {
        // 找到开始按钮
        const startBtn = this.node.getChildByName('StartButton');
        if (startBtn) {
            const button = startBtn.getComponent(Button);
            if (button) {
                button.node.on(Button.EventType.CLICK, () => {
                    console.log('点击了开始按钮');
                    
                    const gm = GameManager.Instance;
                    if (gm) {
                        console.log('GameManager存在，开始关卡1');
                        gm.startLevel(1);

                        console.error('GameManager不存在！');
                        alert('GameManager不存在，请检查场景中是否有GameManager节点');

                }, this);
            }
        }
    }
}
