## 📋 修复内容

### ✅ 修复1：点击食材时在鼠标位置生成

**问题：**
```
点击食材按钮后，食材出现在固定位置
需要移动鼠标才能看到食材
体验不流畅
```

**解决方案：**
```typescript
// 使用触摸事件获取位置
onIngredientClickWithEvent(type, event) {
    // 获取触摸位置
    const touchLocation = event.getLocation();
    
    // 转换为Canvas本地坐标
    const localX = touchLocation.x - canvasSize.width / 2;
    const localY = touchLocation.y - canvasSize.height / 2;
    
    // 立即设置到触摸位置
    this.mouseFollower.setPosition(localX, localY, 0);
}
```

**效果：**
```
✅ 点击食材按钮
✅ 食材立即出现在鼠标/触摸位置
✅ 无需移动鼠标就能看到
✅ 体验流畅自然
```

### ✅ 修复2：拖动时鼠标光标不在素材中心

**问题：**
```
拖动食物时，食物位置和鼠标位置不一致
鼠标在左上角，食物在右下角
偏移累积，越来越不准确
```

**解决方案：**
```typescript
// 1. 拖动开始时计算偏移
onFoodDragStart(event, food) {
    // 计算触摸点相对于食物中心的偏移
    const touchLocalX = touchLocation.x - canvasSize.width / 2;
    const touchLocalY = touchLocation.y - canvasSize.height / 2;
    
    this.dragOffset.x = touchLocalX - bgNode.position.x;
    this.dragOffset.y = touchLocalY - bgNode.position.y;
}

// 2. 拖动时使用偏移
onFoodDragMove(event, food) {
    // 设置食物位置 = 触摸位置 - 偏移
    const newPos = new Vec3(
        touchLocalX - this.dragOffset.x,
        touchLocalY - this.dragOffset.y,
        0
    );
    bgNode.setPosition(newPos);
}
```

**效果：**
```
✅ 拖动开始时记录偏移
✅ 拖动时食物中心跟随鼠标
✅ 偏移保持恒定，不会累积
✅ 精确控制，体验流畅
```
