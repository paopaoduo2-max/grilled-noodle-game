# 快速配置指南 - 10分钟启动游戏

## 🚀 快速步骤

### 步骤1: 打开项目（1分钟）
1. 启动 Cocos Creator 3.x
2. 打开项目：`D:\xm\NewProject_1`
3. 等待编译完成

### 步骤2: 设置MainMenu场景（3分钟）

1. **打开MainMenu.scene**

2. **添加GameManager脚本**
   - 选中层级管理器中的 `GameManager` 节点
   - 在属性检查器点击"添加组件"
   - 搜索 `GameManager`，添加

3. **添加MainMenuUI脚本并绑定**
   - 选中 `Canvas` 节点
   - 添加组件 `MainMenuUI`
   - 绑定属性：
     ```
     Title Label → 拖入 Title 节点
     Level Panel → 拖入 LevelPanel 节点
     Start Btn → 拖入 StartButton 节点
     ```

4. **保存场景** (Ctrl+S)

### 步骤3: 设置PrepareScene场景（3分钟）

1. **打开PrepareScene.scene**

2. **创建TopPanel子节点**
   - 右键 `TopPanel` → 创建 → 创建空节点
   - 重命名为 `LevelNameLabel`
   - 添加组件 `Label`
   - 重复创建：`MoneyLabel`, `TimerLabel`

3. **添加PreparePhaseUI脚本**
   - 选中 `Canvas` 节点
   - 添加组件 `PreparePhaseUI`
   - 绑定所有Label和按钮引用

4. **保存场景**

### 步骤4: 设置CookingScene场景（2分钟）

1. **打开CookingScene.scene**

2. **创建TopInfoPanel子节点**
   - 在 `TopInfoPanel` 下创建4个Label节点：
     - `MoneyLabel`
     - `TimerLabel`
     - `ScoreLabel`
     - `CustomerCountLabel`

3. **添加CookingPhaseUI脚本**
   - 选中 `Canvas`
   - 添加 `CookingPhaseUI`
   - 绑定所有引用

4. **保存场景**

### 步骤5: 设置ResultScene场景（1分钟）

1. **打开ResultScene.scene**
2. **创建ResultPanel子节点**
   - 创建多个Label：TitleLabel、MoneyLabel等
3. **创建ButtonPanel子节点**
   - 创建3个Button：RetryBtn、NextBtn、MenuBtn
4. **添加ResultUI脚本并绑定**
5. **保存场景**

### 步骤6: 设置构建场景列表（30秒）

1. 菜单：`项目 -> 项目设置`
2. 选择"预览运行设置"标签
3. "启动场景"选择：MainMenu
4. 在"参与构建场景"中添加所有4个场景

### 步骤7: 运行测试（30秒）

1. 点击顶部预览按钮 ▶️
2. 或按快捷键 Ctrl+P
3. 游戏应该启动到主菜单

## ⚠️ 常见配置错误

### 错误1: "找不到GameManager"
**解决：** 确保脚本已编译
- 菜单 → 开发者 → 编译脚本

### 错误2: "场景加载失败"
**解决：** 检查场景是否添加到构建列表
- 项目设置 → 参与构建场景

### 错误3: "组件引用为null"
**解决：** 重新绑定组件引用
- 在属性检查器中拖动正确的节点

2. **准备阶段**

   - ✅ 点击"开始营业"进入制作阶段

   - ✅ 8秒后出现第一个客户
   - ✅ 看到客户的耐心条在减少
   - ✅ 点击"开始制作"，食材减少
   - ✅ 5秒后自动完成，获得金币
   - ✅ 等待时间结束或点击"结束营业"

4. **结算界面**
   - ✅ 显示通过/失败
   - ✅ 显示金币、客户数统计
   - ✅ 显示详细统计信息
   - ✅ 可以重试或返回菜单

## 📞 遇到问题？

1. **检查控制台错误**
   - 编辑器底部的控制台
   - 浏览器F12开发者工具

2. **查看日志输出**
   - 所有系统都有日志输出
   - 搜索 `[GameManager]`, `[CustomerManager]` 等

3. **重新编译**
   - 菜单 → 开发者 → 编译脚本
   - 菜单 → 开发者 → 重新加载

4. **查看详细文档**
   - 阅读 `游戏使用说明.md`
   - 里面有更详细的说明
