# ✋ 手动创建测试 - 最简单的方法

## 🎯 目标

在Cocos Creator编辑器中**完全手动**创建一个能显示的场景，不依赖任何工具。

---

## 📋 完整步骤（10分钟）

### 第1步：创建全新场景 ⭐

1. **打开Cocos Creator**

2. **创建新场景：**
   ```
   菜单 → 文件 → 新建场景
   ```

3. **选择场景类型：**
   - 如果有"2D场景"模板，选择它 ✅
   - 如果没有，选择"空场景"

4. **保存场景：**
   ```
   文件 → 保存场景
   名称: TestScene
   路径: assets/Scenes/TestScene.scene
   ```

---

### 第2步：检查/创建Camera 📷

**场景应该自动有Camera，如果没有：**

1. **在层级面板右键 → 创建 → 摄像机**
   - 或者：右键 → 创建节点 → 添加Camera组件

2. **选中Camera节点**

3. **在属性检查器中配置Camera组件：**
   ```yaml
   Camera:
     ☑ 勾选 Visibility 中的:
       ☑ UI_2D        ← 必须勾选！
       ☑ DEFAULT      ← 必须勾选！
     
     Clear Flags: SOLID_COLOR
     Clear Color: 点击颜色选择器
       → R: 51, G: 51, B: 51, A: 255  (深灰色)
     
     Projection: ORTHO
     Ortho Height: 360 (或更大，如720)
     
     Near: 0
     Far: 1000
     
     Priority: 1
   ```

4. **Camera节点的Position:**
   ```
   X: 0
   Y: 0
   Z: 1000  ← 重要！必须在Canvas前面
   ```

---

### 第3步：创建Canvas 🎨

1. **在层级面板右键 → 创建 → 2D对象 → Canvas**
   - 或者：右键 → 创建 → UI组件 → Canvas

2. **Canvas会自动创建，检查配置：**

   **节点属性（在属性检查器顶部）：**
   ```
   Layer: UI 2D  ← 应该自动设置
   
   Position:
     X: 640
     Y: 360
     Z: 0
   ```

   **Canvas组件：**
   ```yaml
   Canvas:
     Camera Component: (可以留空或自动关联)
     Align Canvas With Screen: ☑ 勾选
   ```

   **UITransform组件：**
   ```yaml
   UITransform:
     Content Size:
       Width: 1280
       Height: 720
     
     Anchor Point:
       X: 0.5
       Y: 0.5
   ```

---

### 第4步：创建测试Label 📝

1. **选中Canvas节点**

2. **右键 → 创建 → 2D对象 → Label**
   - 或者：右键 → 创建 → UI组件 → Label

3. **Label节点会自动创建在Canvas下**

4. **选中Label节点，配置：**

   **节点属性：**
   ```
   Name: TestLabel
   
   Position:
     X: 0
     Y: 0      ← 在Canvas中央
     Z: 0
   
   Layer: UI 2D  (自动继承Canvas)
   ```

   **Label组件：**
   ```yaml
   Label:
     String: 测试成功！东北料理王
     
     Font Size: 60
     
     Color: 点击颜色选择器
       → R: 255, G: 200, B: 0, A: 255  (金黄色，更显眼)
     
     Horizontal Align: CENTER
     Vertical Align: CENTER
     
     Use System Font: ☑ 勾选
     Font Family: Arial
   ```

---

### 第5步：保存并预览 ▶️

1. **保存场景：**
   ```
   Ctrl + S (Windows)
   Cmd + S (Mac)
   ```

2. **设置为启动场景：**
   ```
   菜单 → 项目 → 项目设置
   → 预览运行设置
   → 启动场景: 选择 TestScene
   ```

3. **预览：**
   ```
   点击顶部 ▶️ 预览按钮
   或按 Ctrl+P (Windows) / Cmd+P (Mac)
   ```

## 🐛 如果还是黑屏

### 检查1：Camera是否正确

选中Camera节点，确认：
- ✅ Camera组件存在
- ✅ Visibility勾选了 **UI_2D**
- ✅ Visibility勾选了 **DEFAULT**
- ✅ Position Z = 1000 (在Canvas前面)

### 检查2：Canvas是否正确

选中Canvas节点，确认：
- ✅ Layer显示为 **UI 2D**
- ✅ Position大约在 (640, 360, 0)
- ✅ Canvas组件存在

### 检查3：Label是否可见

选中Label节点，确认：
- ✅ Active (勾选框打勾)
- ✅ Label组件Enabled (勾选框打勾)
- ✅ String有文字
- ✅ Font Size够大 (60或更大)
- ✅ Color的Alpha = 255 (不透明)

### 检查4：场景在预览中加载

浏览器控制台(F12)，运行：
```javascript
console.log('Scene:', cc.director.getScene()?.name);
console.log('Canvas:', !!cc.find('Canvas'));
console.log('Label:', !!cc.find('Canvas/TestLabel'));
```

应该输出：
```
Scene: TestScene
Canvas: true
Label: true
```

## 🔧 常见问题修复

### 问题1: Canvas的Layer不是UI 2D

**修复：**
1. 选中Canvas节点
2. 在属性检查器最顶部找到"Layer"下拉框
3. 选择 **UI 2D**

### 问题2: Camera的Visibility没勾选UI_2D

**修复：**
1. 选中Camera节点
2. 找到Camera组件
3. 展开Visibility
4. **必须勾选 UI_2D** 和 DEFAULT

### 问题3: Label的Color是透明的

**修复：**
1. 选中Label节点
2. Label组件的Color
3. 点击颜色选择器
4. 确保A(Alpha) = 255

### 问题4: Camera的Z坐标为0

**修复：**
1. 选中Camera节点
2. Position Z 改为 **1000**
3. 确保Camera在Canvas前面

## 💡 最简化测试

### 如果上述方法还不行，尝试最极简版本：

1. **创建新场景** (使用2D模板)

2. **不做任何修改！**
   - 保持默认的Camera
   - 保持默认的Canvas

3. **只做一件事：**
   - 在Canvas下创建Label
   - Label文字改为"TEST"
   - Label颜色改为红色 (255,0,0)
   - Label字号改为100

4. **立即预览**

**如果这样都看不到，问题可能在：**
- ❌ Cocos Creator版本问题
- ❌ 项目设置问题
- ❌ 显卡/浏览器兼容性问题

## 🎯 为什么要手动创建？

**MCP工具虽然强大，但：**
1. 可能有某些隐藏属性没设置对
2. 场景文件可能有损坏
3. 编辑器需要特定的初始化流程
4. 有些配置在编辑器中更直观

**手动创建的优势：**
1. ✅ 完全可控
2. ✅ 能看到每一步
3. ✅ 编辑器自动处理很多细节
4. ✅ 容易排查问题
