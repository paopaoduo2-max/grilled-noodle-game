# Cocos Creator MCP Server v1.7.3

## 安装说明

1. 将整个文件夹复制到 Cocos Creator 项目的 `extensions` 目录
2. 重启 Cocos Creator 或刷新扩展 (Ctrl+R / Cmd+R)
3. 在扩展菜单中启用 "Cocos MCP Server"
4. 打开扩展面板配置服务器设置

## 使用配置

### Claude Desktop
```json
{
  "mcpServers": {
    "cocos-creator": {
      "type": "http",
      "url": "http://127.0.0.1:3000/mcp"
    }
  }
}
```

### Cursor
```json
{
  "mcpServers": {
    "cocos-creator": {
      "url": "http://127.0.0.1:3000/mcp"
    }
  }
}
```
