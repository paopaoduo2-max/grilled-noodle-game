## 🔧 今日修改详情

### 1. 加工界面增强 (`ProcessingController.ts`)

#### ✅ 已修复的问题
- **错误**: `Can't add component 'cc.UITransform'`
- **原因**: `Sprite` 组件会自动添加 `UITransform`，不能重复添加
- **解决**: 使用 `getComponent(UITransform)` 而不是 `addComponent(UITransform)`

```typescript
// ❌ 错误写法
const transform = node.addComponent(UITransform);

// ✅ 正确写法
const transform = node.getComponent(UITransform);
if (transform) {
    transform.setContentSize(width, height);
}
```

### 2. 库存系统修复 (`InventoryManager.ts`)

#### ✅ 已修复的问题
- **错误**: 场景切换后 `currentLevel` 为 `null`
- **原因**: 
  1. `InventoryManager._instance.destroy()` 导致错误中断 `onLoad`
  2. `ProcessingController.start()` 在 `InventoryManager.onLoad()` 之前执行
- **解决**: 
  1. 移除手动 `destroy()` 调用
  2. 添加 `localStorage` 状态持久化
  3. 延迟 `ProcessingController` 初始化

```typescript
// ✅ 状态持久化
public saveLevelState() {
    const stateData = {
        levelId: this._currentLevel.levelId,
        inventory: inventoryArray,
        // ...
    };
    sys.localStorage.setItem(this.LEVEL_STATE_KEY, JSON.stringify(stateData));
}

// ✅ 延迟初始化
start() {
    this.scheduleOnce(() => {
        this.loadProcessingQueue();
        this.startNextItem();
    }, 0.1);
}
```

## ⚠️ 重要注意事项

### 1. Cocos Creator 组件系统

#### 🚫 常见错误
```typescript
// ❌ 不要重复添加自动组件
const sprite = node.addComponent(Sprite);
const transform = node.addComponent(UITransform); // 错误！Sprite已自动添加

// ❌ 不要手动销毁单例
InventoryManager._instance.destroy(); // 可能导致错误

// ❌ 不要在onLoad中依赖其他组件的初始化顺序
```

#### ✅ 正确做法
```typescript
// ✅ 获取自动添加的组件
const sprite = node.addComponent(Sprite);
const transform = node.getComponent(UITransform);

// ✅ 让引擎管理组件生命周期
InventoryManager._instance = this;

// ✅ 使用延迟初始化处理依赖
this.scheduleOnce(() => {
    // 确保其他组件已初始化
}, 0.1);
```

### 2. 状态管理最佳实践

#### 🔄 场景切换状态保持
```typescript
// ✅ 保存状态到localStorage
public saveLevelState() {
    const stateData = { /* 序列化数据 */ };
    sys.localStorage.setItem(this.LEVEL_STATE_KEY, JSON.stringify(stateData));
}

// ✅ 在场景切换前调用
private goToNextPhase() {
    inventory.saveLevelState(); // 重要！
    director.loadScene('NextScene');
}

// ✅ 在新场景中恢复状态
onLoad() {
    this.loadLevelState();
}
```

#### 📦 单例模式注意事项
```typescript
// ✅ 安全的单例实现
onLoad() {
    if (InventoryManager._instance && InventoryManager._instance !== this) {
        // 复制状态，不要destroy
        if (InventoryManager._instance._currentLevel) {
            this._currentLevel = InventoryManager._instance._currentLevel;
        }
    }
    InventoryManager._instance = this;
}
```
