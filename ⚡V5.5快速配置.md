# ⚡ V5.5 快速配置指南

## 🎯 2分钟完成配置！

---

## 步骤1：绑定Serve Button（重要！）

```
1. 打开 Cocos Creator
2. 打开 CookingScene 场景
3. 选中 Canvas 节点
4. 在属性检查器找到 CookingControllerV2 组件
5. 找到 "Serve Button" 属性
6. 从层级管理器拖入 ServeButton 节点
7. 保存场景（Ctrl+S）
```

### 检查清单
```
✅ Serve Button: ServeButton（不是null）
✅ Packing Box: PackingBox（不是null）
✅ Trash Bin: TrashBin（不是null）
```

---

## 步骤2：运行测试

```
1. 点击运行按钮
2. 查看控制台

应该看到：
✅ [CookingControllerV2] ✅ 出餐按钮已设置

如果看到：
⚠️ [CookingControllerV2] ⚠️ 出餐按钮未绑定！
   → 返回步骤1，检查绑定
```

---

## 步骤3：测试拖动

```
1. 制作烤冷面
2. 拖动到右侧金黄色打包盒
3. 松手
4. 查看控制台

应该看到：
✅ [Debug] 🔵 hitTest: ✅✅✅ 成功！
✅ [Debug] ✅✅✅ 在范围内！
✅ 📦 食物已打包！
✅ 出餐按钮变绿
```

---

## ❌ 如果失败

### 问题1：出餐按钮未绑定
```
解决：
1. 检查场景中是否有ServeButton节点
2. 如果没有，检查是否被删除
3. 重新绑定到CookingControllerV2组件
```

### 问题2：hitTest失败
```
检查：
- 控制台是否显示"hitTest: ❌ 失败"
- 是否使用了备用检测
- 备用检测是否通过

如果都失败：
- 发送完整控制台日志
- 告诉我距离和半径数值
```
