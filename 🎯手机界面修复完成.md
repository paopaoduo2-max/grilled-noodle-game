✅ **屏幕尺寸自动检测**

```typescript
// 获取屏幕宽度和高度，用于响应式设计
const canvasTransform = canvas.parent.getComponent(UITransform);
```

✅ **响应式布局切换**
```typescript
// 手机竖屏（<768px）：垂直排列
if (isMobile) {
    layout.type = Layout.Type.VERTICAL;  // 切换为竖直布局
    layout.spacingY = 15;                // 适当间距
}

// 桌面水平（>=768px）：水平排列
else {
    layout.type = Layout.Type.HORIZONTAL;  // 保持水平布局
    layout.spacingX = 20;                 // 更大间距
}
```

✅ **动态字体大小**
```typescript
// 手机显示32px，桌面显示48px
const fontSize = this._screenWidth < 768 ? 32 : 48;
this.titleLabel.fontSize = fontSize;
```

✅ **动态按钮宽度**
```typescript
// 手机按钮宽度 = 屏幕宽度 - 60px
const btnWidth = isMobile ? this._screenWidth - 60 : 400;
```

## 📊 响应式断点

系统使用以下断点进行响应式设计：

| 屏幕宽度 | 类别 | 布局 | 字体大小 |
|---------|------|------|--------|
| < 768px | 手机 | 竖直 | 32px |
| >= 768px | 桌面 | 水平 | 48px |

可在 `MainMenuUI.ts` 中修改 `768` 这个断点值：
```typescript
const isMobile = this._screenWidth < 768;  // ← 可修改此值
```

## 💡 常见问题

### Q: 关卡按钮还是不显示？
**A:** 检查控制台日志，查看是否有错误。如果 `LevelPanel` 显示为 `false`，说明节点查找失败，可能是节点名称不正确。

### Q: 布局切换不生效？
**A:** 需要重新编译脚本。按 `Ctrl+Shift+B` 或在菜单中选择"开发者"→"编译脚本"。

### Q: 字体大小还是太小/太大？
**A:** 修改 `MainMenuUI.ts` 中的这一行：
```typescript
const fontSize = this._screenWidth < 768 ? 32 : 48;
//                                         ↑↑  ↑↑
//                         手机     桌面
```

### Q: 如何在平板上显示？
**A:** 平板通常宽度在 768-1024px 之间，会使用水平布局。如果想要特殊处理，可以添加更多断点：
```typescript
if (this._screenWidth < 600) { /* 手机 */ }
else if (this._screenWidth < 1024) { /* 平板 */ }
else { /* 桌面 */ }
```

## 📞 技术细节

### 响应式设计原理

1. **屏幕检测**：从 Canvas 节点获取实际屏幕尺寸
2. **条件判断**：根据宽度判断设备类型
3. **动态配置**：修改 Layout 组件的属性
4. **即时生效**：无需手动刷新，自动应用

### 相关组件

- `UITransform` - 控制节点大小和位置
- `Layout` - 控制子节点的排列方式
- `Label` - 文字显示，支持动态字体大小
