## 事件类型
| 类型 | 说明 |
|------|------|
| `normal` | 普通事件，立即生效 |
| `production` | 制作挑战，限时制作指定数量 |
| `chain` | 链式事件，N天后触发后续 |
| `delivery` | 需制作交付才能完成 |
