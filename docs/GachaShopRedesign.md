# 扭蛋店重新设计方案（参考彩票站“3 连不中必中”）

## 目标
- 最大化刺激与留存：每 3 次未命中 SR 及以上自动触发必中，形成“再来一发就回本”的心理钩子。
- 让玩家“眼见为实”：强化投币—拉杆—掉球—开蛋—爆光的物理感与节奏感。
- 保持经济可控：单次消费占玩家持币的 10%~25%，多档位覆盖不同风险偏好。

## 核心设计理念
- 借用彩票站成功机制：复制“3 连不中必中”的短周期保底，再叠加 8 抽软保底、10 抽硬保底的长周期保底。
- 让过程更爽、结果更炸：动画比概率更能留人，稀有结果用爆光+慢放+停顿。
- 给出“下一发更划算”的理由：动态概率提示、累计进度条、连击奖励都可视化。

## 核心循环（玩家视角）
1) 选机与档位：普通 10 / 高级 20 / 豪华 30（保持现有三档，UI 显眼标价与稀有率差异）。  
2) 选模式：单抽、5 连、10 连（批量抽默认播放首末两颗，其他快放）。  
3) 支付与动作：投币 → 拉杆/按钮 → 球体震动 → 胶囊滚落 → 打开。  
4) 结算：弹窗展示稀有度、奖励金币/道具，连击/保底进度前置显示。  
5) 诱导下一发：进度条已到 2/3 连不中，显著提示“再来必中 SR”；软保底接近时提示“下次提升稀有率 +X%”。

## 概率与保底设计
- 基础概率（单机）  
  - 普通机：N62 / R25 / SR10 / SSR2 / UR1  
  - 高级机：N52 / R30 / SR15 / SSR2 / UR1  
  - 豪华机：N43 / R35 / SR20 / SSR1 / UR1  
- 短周期保底（复制彩票站体验）  
  - 连续 3 抽未出 SR 及以上：下一抽必出 SR（最低保底）。  
  - UI：右上角“厄运计数 2/3，下发必中 SR”红色提示。  
- 长周期保底  
  - 软保底：第 8 抽起每抽稀有率 +2%（均摊到 SR/SSR/UR）。  
  - 硬保底：第 10 抽必出 SR+，并将概率向 SSR 倾斜（SR→70%，SSR→25%，UR→5%）。  
- 连击（Lucky Streak）  
  - 连续 3 次抽出 SR+：下一抽金币奖励 ×1.2；连击 UI 金色闪烁。  
- 追击（Double Down）  
  - 抽后弹出“追击”按钮：花当前档位 50% 价格再抽 1 次，继承当前保底计数，动画加速播放。

## UI 结构（移动竖屏优先）
- 顶部：金币、返回、帮助（展示保底/概率公式），右侧“3 连不中必中”红色徽标。  
- 主视窗：居中大扭蛋机（带玻璃、扭杆、投币口、出球口）。  
- 底部按钮区：  
  - 左侧档位 Tab（普通/高级/豪华，颜色与价位对应）。  
  - 中部抽取按钮（单抽、5 连、10 连），显示价格与折扣（10 连=9.5 折）。  
  - 右侧“追击”按钮（出现条件：上一抽未命中 SR+）。  
- 进度与历史：  
  - 保底进度条 1：3 连不中计数（0/3，必中 SR）。  
  - 保底进度条 2：软/硬保底计数（0/8/10）。  
  - 最近 5 次弹幕式滚动（稀有度+奖励），SSR/UR 高亮并短震动。  
- 图鉴入口：显示已收集套装进度，鼓励收集。

## 动画与特效节奏
- 投币：金币从钱包飞入投币口，金属落币声。  
- 拉杆：杆子压下回弹，机体轻微摇晃。  
- 震动：玻璃球内彩球随机滚动 0.4s。  
- 掉落：胶囊沿轨道滚出，落地 1~2 次弹跳。  
- 开蛋：胶囊上盖旋转弹开，先出稀有度光效，再展示奖励卡片。  
- 稀有光效：  
  - SR：金色扫光 + 粒子  
  - SSR：粉金爆光 + 轻微屏幕震动  
  - UR：紫金爆光 + 全屏闪 + 慢动作 0.3s  
- 批量抽：首颗按完整流程，2~(n-1) 颗快速闪过，最后 1 颗慢放；若有 UR，自动慢放到该颗。

## 交互与提示
- 悬停/点按档位 Tab：弹出本机概率和奖池预览。  
- 保底即将触发时，抽取按钮添加“必中”/“概率提升”文案。  
- 结果弹窗提供“再抽一次”和“换机试试”双按钮。  
- 可在设置中切换“快速开蛋”与“完整动画”。

## 收集与循环
- 图鉴套装：每台机 8~10 件，集齐送限定头像框/一次 UR 直取券。  
- 每日任务：抽 10 次、触发 1 次保底、获得 1 次 SR+；完成后赠送 1 次普通机单抽。  
- 限时机：每日刷新 1 次主题机（限时概率或限定奖励），鼓励回访。

## 数值与经济建议
- 单抽价格继续 10/20/30；10 连 9.5 折。  
- 金币产出参考 GachaConfig：N 5~10，R 15~30，SR 40~80，SSR 100~200，UR 300~500，可在豪华机上整体 +10%。  
- 限制：每日免费券≤2，防止无限刷。

## 与现有代码的接入点
- `assets/Scripts/Game/GamblingShops/GachaMachine/GachaConfig.ts`  
  - 对齐短周期保底：`unluckyStreakThreshold = 3`，`unluckyStreakGuaranteedRarity = SR`。  
  - 软/硬保底参数：`softPityStart = 8`，`hardPity = 10`，`softPityIncrement = 2`。  
  - 如需追击功能，可在配置中增加 `chaseCostMultiplier = 0.5`，并在 UI 上动态显示。  
- `GachaMachineController.ts`  
  - UI：新增/强化双进度条、批量抽首尾慢放逻辑、追击按钮状态机。  
  - 动画：拉杆、滚落、开蛋、稀有光效按“动画与特效节奏”实现。  
  - 提示：在抽取前根据计数更新按钮文案（必中/概率提升），结果弹窗增加“再抽一次/换机试试”。  
- `Factory/GachaUIFactory.ts` / `Components`  
  - 机器外观：玻璃球内加彩球粒子，投币口/出球口加发光描边；胶囊颜色与稀有度挂钩。  
  - 批量抽：支持跳过按钮，或在 `AnimationTimings` 中区分首末与中间速度。

## 验收清单
- UI：双进度条、档位 Tab、抽取区、历史弹幕、图鉴入口完整可见。  
- 动作：投币→拉杆→滚落→开蛋→结算全链路无卡顿，批量抽支持快放。  
- 数值：3 连不中必中 SR，8 抽起软保底，10 抽硬保底；概率/奖励与 UI 显示一致。  
- 反馈：SR/SSR/UR 具有差异化光效与震动；按钮文案能提示保底状态。  
- 运营：每日任务与限时机可配置，经济不超发。
