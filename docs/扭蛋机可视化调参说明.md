# 扭蛋机可视化调参说明

本说明用于调节扭蛋机素材与玻璃范围，所有参数在编辑器 Inspector 中完成。

## 组件位置
- 节点：包含 `GachaMachineController` 组件的扭蛋机入口节点
- 组件路径：`assets/Scripts/Game/GamblingShops/GachaMachine/GachaMachineController.ts`

## 玻璃范围（胶囊活动区）
- `globeOffset`：玻璃球中心点偏移（X/Y）
- `globeRadiusOverride`：玻璃球半径（>0 时生效，0 使用默认半径）

调参建议：
1. 先设置 `globeRadiusOverride` 为接近素材玻璃内缘的半径。
2. 用 `globeOffset` 对齐玻璃中心，确保胶囊不会穿出玻璃边缘。

## 前景玻璃 / 后景机身贴图
- 打开 `useLayeredSprites`
- 后景机身贴图：
  - `normalMachineBackSprite`
  - `premiumMachineBackSprite`
  - `luxuryMachineBackSprite`
- 前景玻璃贴图：
  - `normalGlassSprite`
  - `premiumGlassSprite`
  - `luxuryGlassSprite`
- 后景机身偏移/缩放：`machineBackOffset` / `machineBackScale`
- 前景玻璃偏移/缩放：`glassFrontOffset` / `glassFrontScale`
- 需要隐藏矢量玻璃时：勾选 `hideVectorGlassWhenLayered`
- 需要隐藏矢量机身/拉杆时：勾选 `hideVectorPartsWhenSprite`

## 单张机身贴图（不分层）
- 关闭 `useLayeredSprites`
- 打开 `useMachineSprite`
- 贴图：`normalMachineSprite` / `premiumMachineSprite` / `luxuryMachineSprite`
- 偏移/缩放：`machineSpriteOffset` / `machineSpriteScale`

## 回退与安全建议
- 调参前记录当前数值（或先保存一份场景副本）。
- 需要回退时：
  - 关闭 `useLayeredSprites` 和 `useMachineSprite`
  - `globeRadiusOverride` 设为 0
  - 所有偏移/缩放恢复为默认值（0 或 1）

## 常见问题
- 胶囊超出玻璃范围：减小 `globeRadiusOverride` 或上移 `globeOffset.y`
- 胶囊偏离玻璃中心：微调 `globeOffset.x/y`
- 贴图遮挡顺序不对：确保前景玻璃使用 `glassFrontSprite`，后景机身使用 `machineBackSprite`
