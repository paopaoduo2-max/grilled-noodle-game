## 📊 效果说明

| 符号 | 含义 |
|------|------|
| 💰 money | 金币变化 |
| 🔥 heat | 热度变化 |
| 👥 customerRate | 客流倍率（>1增加，<1减少）|
| ⏰ customerRateDays | 客流效果持续天数 |
| 🎯 successRate | 成功概率（%）|
| 🏭 production | 制作挑战（需要完成指定数量）|
| 🔗 chain | 链式事件（几天后触发后续效果）|

## 🔧 修改指南

### 修改事件效果
1. 打开 `assets/Scripts/Game/RandomEventSystemV2.ts`
2. 找到对应事件ID（如 `L01`）
3. 修改 `effect` 对象中的数值

### 添加新事件
```typescript
{
    id: 'L26',                    // 唯一ID
    name: '新事件名称',            // 显示名称
    icon: '🎉',                   // 图标
    timeSlot: 'lunch',            // 时段
    type: 'normal',               // 类型
    sender: { 
        name: '发起人名', 
        icon: '👤', 
        role: '身份' 
    },
    description: '事件描述',
    optionA: { 
        text: '选项A文字', 
        emoji: '✅',
        effect: { money: 100, heat: 20 }
    },
    optionB: { 
        text: '选项B文字', 
        emoji: '❌',
        effect: { heat: -10 }
    }
}
```
