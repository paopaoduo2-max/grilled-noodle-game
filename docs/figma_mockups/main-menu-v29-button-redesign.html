按钮重设计 V29

主按钮与设置按钮统一成“木框 + 招牌底 + 中央亮带”的同一套语言

视觉方向

按钮不再只是浅色圆角条，而是更像店招部件。主按钮做成厚木框包裹的暖金招牌，保留中间亮带给文字；设置按钮缩成同语言的小牌。

设计要点

按钮不再漂白 主体回到暖金与木色，强调“店招”和“牌匾”感，而不是奶白卡片。
主次状态更明确 三颗主按钮保持同一家族，用左侧点缀色区分开始、继续、退出，但不打散整体风格。
文字承载区更稳定 中间亮带更整块，保证 Cocos 的 Label 叠上去时不脏、不压背景。
设置按钮同语言 设置入口不再像另一套组件，尺寸更小，但视觉语法与主按钮一致。
