# Codex 多角色并行启动说明（单账号）

## 1) 前置
- 只需要登录一次 Codex 账号。
- 在项目根目录运行，确保加载本目录 `.codex/` 配置。

## 2) 启动
```powershell
cd /d D:\xm\NewProject2
codex
```

## 3) 首条指令（可直接粘贴）
```text
请按以下角色并行执行今天目标：
- planner：拆解并行任务与合并顺序
- gameplay：实现经济与解锁（初始1000，地图2解锁2000，首批设备免费）
- story：老师傅教程对话（非强制，可做错重做）
- scene_mcp：关卡选择改城市地图解锁表现，旧Level2-6入口下线
- qa：全链路冒烟与风险报告

要求：
1) 每个角色只负责自己范围
2) 输出各自 PR 标题、改动摘要、回归路径
3) 最后给出合并顺序
```

- 主流程闭环可跑通
