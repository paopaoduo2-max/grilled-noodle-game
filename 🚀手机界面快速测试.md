## 🔧 常见问题快速解决

### 问题1：关卡按钮显示不出来

**检查步骤：**
```
1. 按 F12 打开控制台
2. 查看是否有红色错误
3. 搜索 [MainMenuUI] 关键字
4. 检查是否显示 "找到LevelPanel: false"
```

**解决方案：**
- 如果是 false，说明节点名称错误
- 在 Cocos Creator 中检查 LevelPanel 节点是否存在
- 删除场景中的MainMenuUI组件，重新添加

### 问题2：字体太小看不清

```
文件：assets/Scripts/UI/MainMenuUI.ts
找到这一行（约第103行）：
const fontSize = this._screenWidth < 768 ? 32 : 48;

修改为：
const fontSize = this._screenWidth < 768 ? 40 : 56;
                                         ↑↑  ↑↑
                                    手机  桌面

然后重新编译（Ctrl+Shift+B）
```

### 问题3：布局还是是水平的，不是竖直的

```
检查你的屏幕宽度是否真的小于768px

在浏览器控制台运行：
window.innerWidth

如果显示 > 768，那就是宽度问题
试试：
- 最大化浏览器窗口（通常>1024px，显示水平）
- 缩小浏览器宽度到500px以下
- 使用开发者工具的设备模拟（Ctrl+Shift+M）
```

### 问题4：点击按钮没反应

```
1. 确认按钮是否显示
2. 按 F12 查看控制台
3. 搜索 [MainMenuUI] setupButtons
4. 检查是否成功绑定了事件
5. 尝试点击不同位置的按钮
```
