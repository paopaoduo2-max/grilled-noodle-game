## 🔄 替换步骤

1. 准备好对应尺寸的图片
2. 保持文件名不变
3. 放入对应文件夹
4. 重新运行游戏

## 💡 设计建议

- 烤盘使用金属质感，有油渍效果更真实
- 火焰动画要有明显的跳动感
- 食材状态变化要清晰（生→熟→焦）
- 工作区域边框要明显，便于玩家识别
- 动画帧之间过渡要自然
