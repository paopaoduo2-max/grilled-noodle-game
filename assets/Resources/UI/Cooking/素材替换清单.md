## 🔄 替换方式

### 方式1: 直接替换文件
1. 准备好对应尺寸的图片
2. 保持文件名与清单一致
3. 放入 `assets/resources/UI/Cooking/` 对应文件夹
4. Cocos Creator 会自动刷新

### 方式2: 代码中加载
```typescript
// 在 CookingControllerV2.ts 中
resources.load('UI/Cooking/bg_cooking_main/spriteFrame', SpriteFrame, (err, spriteFrame) => {
    if (spriteFrame) {
        bgSprite.spriteFrame = spriteFrame;
    }
});
```

### 方式3: 修改节点属性
1. 在 CookingScene.scene 中选中对应节点
2. 在属性面板中拖入新的 SpriteFrame
3. 保存场景
