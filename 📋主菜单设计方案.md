# 主菜单系统设计方案

## 一、主菜单UI改进方案

### 1.1 移除LevelPanel
**现状**：主菜单显示关卡选择面板
**改进**：主菜单作为游戏入口，简洁明了，移除关卡选择
**原因**：
- 主菜单应该简洁，只保留核心功能入口
- 关卡选择可以在点击"开始游戏"后进入专门的关卡选择界面
- 提升用户体验，避免信息过载

### 1.2 主菜单新增功能

#### **主要按钮（垂直排列）**
1. **开始游戏** - 进入关卡选择或直接开始
2. **继续游戏** - 加载最近的存档（如果有）
3. **读取存档** - 打开存档列表
4. **设置** - 打开设置面板
5. **退出游戏** - 退出（浏览器环境显示提示）

## 二、设置系统设计

### 2.1 设置面板功能

#### **音效设置**
- 🎵 背景音乐开关
- 🎵 背景音乐音量（0-100%）
- 🔊 音效开关
- 🔊 音效音量（0-100%）

#### **画质设置**
- 🎨 画质等级（低/中/高）
  - 低：关闭粒子效果、降低贴图质量
  - 中：标准效果
  - 高：全部特效开启
- 📱 帧率限制（30/60/不限制）

#### **游戏设置**
- 🌐 语言选择（中文/English）
- ⏱️ 显示FPS（开/关）
- 🎮 自动保存（开/关）
- 📊 显示提示（新手提示开关）

#### **控制设置**
- 🖱️ 鼠标灵敏度（1-10）
- ⌨️ 快捷键显示（开/关）
- 📱 触摸反馈（开/关，移动端）

### 2.2 设置数据结构

```typescript
interface GameSettings {
    // 音效
    audio: {
        bgmEnabled: boolean;
        bgmVolume: number;      // 0-100
        sfxEnabled: boolean;
        sfxVolume: number;      // 0-100
    };
    
    // 画质
    graphics: {
        quality: 'low' | 'medium' | 'high';
        frameRate: 30 | 60 | 0;  // 0表示不限制
    };
    
    // 游戏
    game: {
        language: 'zh' | 'en';
        showFPS: boolean;
        autoSave: boolean;
        showTips: boolean;
    };
    
    // 控制
    controls: {
        mouseSensitivity: number;  // 1-10
        showHotkeys: boolean;
        touchFeedback: boolean;
    };
}
```

## 三、存档系统设计

### 3.1 存档功能

#### **自动存档**
- 每个关卡结束后自动保存
- 每次购买食材后保存
- 退出游戏前自动保存
- 最多保留5个自动存档槽位（循环覆盖）

#### **手动存档**
- 玩家可以在任何时候手动保存
- 最多10个手动存档槽位
- 每个存档显示：
  - 存档名称（可自定义）
  - 保存时间
  - 当前关卡
  - 金币数量
  - 游戏进度（已完成关卡数）
  - 缩略图（可选）

### 3.2 存档数据结构

```typescript
interface SaveData {
    // 基本信息
    saveId: string;              // 存档ID
    saveName: string;            // 存档名称
    saveType: 'auto' | 'manual'; // 存档类型
    timestamp: number;           // 保存时间戳
    
    // 游戏进度
    playerData: {
        totalMoney: number;      // 总金币
        currentLevel: number;    // 当前关卡ID
        unlockedLevels: number[];  // 已解锁关卡
        completedLevels: number[]; // 已完成关卡
    };
    
    // 统计数据
    stats: {
        totalPlayTime: number;   // 总游戏时长（秒）
        totalOrders: number;     // 总订单数
        totalCustomers: number;  // 总顾客数
        superGoodReviews: number;
        goodReviews: number;
        badReviews: number;
    };
    
    // 成就数据
    achievements: {
        unlockedAchievements: string[];  // 已解锁成就ID
    };
    
    // 设置
    settings: GameSettings;
}
```

## 四、游戏架构查漏补缺

### 4.1 核心系统缺失（高优先级）

#### **1. 音频系统 ⭐⭐⭐⭐⭐**
**现状**：无音频管理
**需求**：
- 背景音乐管理器（循环播放、音量控制、淡入淡出）
- 音效管理器（同时播放多个音效、音效池）
- 与设置系统联动

**建议实现**：
```typescript
class AudioManager {
    - playBGM(clip: string, loop: boolean, volume: number)
    - stopBGM(fadeOut: boolean)
    - playSFX(clip: string, volume: number)
    - setMasterVolume(volume: number)
    - setBGMVolume(volume: number)
    - setSFXVolume(volume: number)
}
```

#### **2. 教程系统增强 ⭐⭐⭐⭐⭐**
**现状**：有基础教程但不够完善
**需求**：
- 分步引导（高亮显示、遮罩效果）
- 可跳过的教程
- 教程进度保存
- 游戏内提示系统

#### **3. 关卡选择界面 ⭐⭐⭐⭐⭐**
**现状**：关卡选择在主菜单中
**需求**：
- 独立的关卡选择场景
- 显示关卡预览、目标、奖励
- 关卡难度指示
- 关卡解锁动画

#### **7. 每日任务系统 ⭐⭐⭐**
**需求**：
- 每日任务刷新
- 任务进度追踪
- 任务奖励
